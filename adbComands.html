ADB Commands

Lista de comandos adb que mais utilizamos. Para utilizar esses comandos, é necessários instalar o sdk manager, e seta-los nas variáveis de sistema do seu computador. No programa QATools existem outros comandos e opções disponíveis, e você pode chegar todos os .bat na rais do programa.

Clique para abrir o tutorial.
              

Command: Install

Função: Instala qualquer aplicativo, a partir de um .apk em seu computador
Estrutura: adb install (path do apk + nome + extensão)
Exemplo: adb install C:\Users\rebec\Downloads\pwp.apk

Download .bat

Command: Uninstall

Função: Desinstala aplicativos no celular, para utilizar você deve passa o nome do package do aplicativo
Estrutura: adb uninstall (package name)
Exemplo: adb uninstall com.psafe.powerpro

Download .bat

Command: Upgrade

Função: Faz o upgrade de aplicativos a partir de um mais recente no seu computador
Estrutura: adb uninstall -r (package name)
Exemplo: adb uninstall -r com.psafe.powerpro

Download .bat

Command: Clear Data

Função: Limpa o cache de um aplicativo, para utilizar você deve passa o nome do package do aplicativo.
Estrutura: adb shell pm clear (package name)
Exemplo: adb shell pm clear com.psafe.powerpro

Command: Screenshot

Função: Tira uma foto da tela do seu celular, o bat em download também faz a copia do arquivo para o pc. O exemplo mostra apenas o comando para tirar a foto da tela
Estrutura: adb shell screencap -p
Exemplo: adb shell screencap -p

Estrutura: adb shell screencap -p
Exemplo: adb shell screencap -p

Command: LogCat

Função: Captura os logs de um app no celular, deve ser terminado o package name do aplicativo
Estrutura: adb logcat -d (package name)
Exemplo: adb logcat -d com.psafe.powerpro

Command: Battery change status

Função: Muda a quantidade de bateria restante no celular, deve ser determinado qual o novo valor da bateria
Estrutura: adb shell dumpsys battery set level (valor desejado da bateria)
Exemplo: adb shell dumpsys battery set level 10

Download .bat

Command: Battery resete status

Função: Reseta todas as configurações setadas anteriormente para bateria. Restaurando o valor original do telefone
Estrutura: adb shell dumpsys battery reset
Exemplo: adb shell dumpsys battery reset

Command: Battery mascara USB

Função: Simula que o celular esta desplugado do usb, mesmo com o usb ligado
Estrutura: adb shell dumpsys battery set usb 0
Exemplo: adb shell dumpsys battery set usb 0

Command: Device pareado

Função: Verifica todos os celulares conectados no pc. Caso seja exibido o numero e modelo do celular, significa que os comandos adb estão funcionando corretamente
Estrutura: adb devices
